# ds-win32-x64

Optional platform package directory for the DS CLI native binary.

**This fork is built from source** — see the repository root
[README.md](../../../../../README.md) and [DEEPSEEK.md](../../../../../DEEPSEEK.md).

```bash
cargo build -p ds-pager-bin --release
install -m 755 target/release/ds-pager ~/.local/bin/ds
```
