# Contributing

This public tree is published for **source transparency and local builds**
under the Apache License 2.0 (see [`LICENSE`](LICENSE)).

It is a re-vendored, rebranded fork of Grok Build (DS Build + DeepSeek).
**External pull requests and unsolicited patches are not accepted.**

## Security reports

Please report security issues through the process described in
[`SECURITY.md`](SECURITY.md). Do not open a public issue for vulnerabilities.
